## Sprints

### Sprint 1

Foco em estruturação inicial do projeto, requisitos, diagramas e protótipos.

### Sprint 2

Foco na integração das funcionalidades principais com o banco de dados:

- Cadastro de serviços;
- Cadastro de barbeiros;
- Agendamento;
- Visualização de agendamentos;
- Configuração do GitHub Actions.

### Sprint 3

Foco em:

- Permissões de usuários;
- Melhorias de UI/UX;
- Dashboard administrativo;
- Gerenciamento de usuários;
- Agenda do barbeiro.
